En esta Aplicación vamos a explorar tanto como podamos la creación de activities

-SavingRestoringActivity:
Muestra como guardar y recuperar el estado de una activity

-ButtonsAndTextFieldsActivity:
Muestra una activity con un Botón y un texto. Ve a activity_buttons_and_text_fields.xml
	- Ademas muestra como añadir un evento al botón
	
	
-NavegacionEntreActivities:
Muestra la navegación de una actividad en otra. Va a la clase NavegacionEntreActivities.java
para ver como se implementa
